## 🚀 Quick Start

### Prerequisites
- Node.js (v18+)
- PostgreSQL (v14+)
- npm หรือ yarn

### Installation

```bash
# 1. Clone repository
git clone <repository-url>
cd apexops_i

# 2. Install backend dependencies
cd server
npm install
cp .env.example .env
# แก้ไข .env ตามการตั้งค่าของคุณ

# 3. Install frontend dependencies
cd ../client
npm install

# 4. Start PostgreSQL service
# (ตรวจสอบว่า PostgreSQL ทำงานอยู่)

# 5. Start backend server
cd ../server
npm run dev

# 6. Start frontend (ใน terminal ใหม่)
cd ../client
npm run dev
```

### Access
- Frontend: http://localhost:5173
- Backend API: http://localhost:3000
